# nodejs-web-vite-vue-typescript-template

## Description
A vue template for the vite packager.

## Tech stack
- npm
- vite
- vue
- typescript

## Docker stack
- node

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
